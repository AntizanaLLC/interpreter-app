# Interpreter - Voice Translator

Real-time voice translation for iPhone. Speak — and hear the translation instantly.

## Features

- **On-device speech recognition** — Whisper AI runs entirely on your iPhone
- **10 languages** — English, Spanish, Russian, French, German, Portuguese, Polish, Dutch, Italian, Maltese
- **Conversation Mode** — face-to-face interpretation for two people
- **Works offline** — no internet required after initial setup
- **Privacy first** — your voice never leaves your device

### Common issues

**The app is downloading on first launch**
Interpreter downloads the Whisper speech recognition model (~74 MB) on first launch. Make sure you have a WiFi connection.

**Translation is not accurate**
For best results, speak clearly and close to the microphone. Background noise can affect accuracy.

**A language is not being detected**
Try selecting the source language manually instead of using Auto-detect.
